E. coli flux balance analysis performs steady state flux balance analysis on E. coli core metabolism. It models the metabolism of an E. coli cell using a network of reactions. The flux, or amount of reactant converting to product, of each link in this reaction network is determined using an assumption of steady state. Steady state assumes products and reactants don't accumulate in the cell and there are no concentration-dependent effects on metabolism. These MATLAB scripts find how active each metabolic pathway is and allows metabolic analysis of genetic knockouts on E. coli.

STDinfo implements privacy-preserving data analytics for medical data. It is a Python script that infers transmission risk from 2 medical test reports. It is applied to sexually transmitted diseases (STDs) to find transmission risk between 2 individuals based on their STD panel test reports while keeping the reports private. Performing data analytics while keeping underlying data private has uses in healthcare data analyses, semiconductor fabrication optimization and privacy in large language models. If you'd like for us to explore these or other applications to implement STDinfo in your organization, we are currently running a free pilot till May 2024.

Automeet is a bot to attend video meetings. It keeps track of a timetable to attend and exit Google meetings within given times. Graphical user interface is manipulated using basic computer vision to enter and exit a video conference just as a human would. This bot may be reconfigured for GUI manipulation using computer vision or making summaries of video meetings.
